# Cybersecurity Log Analyzer

This project analyzes authentication logs to identify suspicious login activity,
including repeated failed login attempts that may indicate brute-force attacks.

## Features
- Detects multiple failed login attempts
- Flags suspicious IP addresses
- Simple and ethical security monitoring

## Technologies Used
- Python 3

## Example Output
Suspicious IP Detected: 192.168.1.25 (3 failed attempts)
